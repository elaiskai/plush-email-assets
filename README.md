# Plush Email Assets

Hero images, product shots, and composites for Plush email campaigns.

Used by email HTML via jsDelivr CDN:
```
https://cdn.jsdelivr.net/gh/elaiskai/plush-email-assets@main/heroes/[filename].png
```

## Structure
- `/heroes/` — Gemini-generated campaign hero images
- `/products/` — product shots from plush.lt
- `/composites/` — multi-product lifestyle / bundle images
